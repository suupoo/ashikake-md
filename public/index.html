使い方ガイド

基本的な使い方

エディターにマークダウン形式でドキュメントを入力すると、プレビューがリアルタイムで表示されます。内容は自動保存されます。

PC版： 左側のエディターと右側のプレビューが同時に表示されます。

スマホ版： エディターの下にある「エディター」「プレビュー」ボタンで切り替えができます。

マークダウン記法

  • # 見出し1 - 大見出し
  • ## 見出し2 - 中見出し
  • ### 見出し3 - 小見出し
  • - リスト項目 - 箇条書き
  • **太字** - 太字
  • *斜体* - 斜体

テーブル

表を作成するには以下の形式を使用してください：

| 項目 | 内容 |
|------|------|
| スキル | JavaScript |
| 経験年数 | 3年 |

HTMLタグの使用

テキスト配置などでHTMLタグも使用できます：

  • <div class="text-center">中央揃え</div>
  • <div class="text-right">右揃え</div>
  • <div class="text-justify">両端揃え</div>

Mermaid図表

フローチャートやタイムラインを描画できます：

```mermaid
timeline
    title プロジェクト進行
    2023-01 : 設計開始
    2023-03 : 開発開始
    2023-06 : リリース
```

改ページ

印刷時に改ページしたい箇所で「改ページ挿入」ボタンを押すか、---を入力してください。

マルチファイル機能

複数のファイルを管理して効率的に作業できます：

PC版：

  • 「＋」ボタンで新しいファイルを作成
  • ファイルタブをクリックして切り替え
  • ファイル名をダブルクリックで名前を変更
  • ×ボタンでファイルを削除

スマホ版：

  • ハンバーガーメニュー（≡）からファイル一覧を表示
  • 「新規ファイル」ボタンで新しいファイルを作成
  • ファイル名をタップして切り替え

操作方法

PC版：

  • ヘッダー右上の「？」ボタンでヘルプを表示
  • ファイルタブが上部に表示されます
  • エディターとプレビューが同時に表示されます

スマホ版：

  • ヘッダー右上の「？」ボタンでヘルプを表示
  • ハンバーガーメニュー（≡）でファイル操作
  • エディター下部のボタンで表示を切り替え

キーボードショートカット

  • Ctrl + S - 手動保存
